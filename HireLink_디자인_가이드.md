# HireLink 디자인 레퍼런스 & 스타일 가이드

> 한 줄 컨셉: **"차분하고 큐레이션된 에디토리얼"** — 아이비리그 대학 소개 책자 + 프리미엄 금융/컨설팅 브로셔의 톤을 웹으로 옮긴 느낌.
> 학생이 "여긴 진지하게 관리되는 곳이구나"라고 느끼고, 기업이 "신뢰할 수 있는 인재풀"이라고 읽게 만드는 것이 목표.

---

## 1. 현재 디자인이 "AI 느낌"인 진단

현재 시안에서 AI 느낌을 만드는 주된 원인들:

- **카드 남용** — 섹션마다 라운드 카드 + soft shadow. 모든 정보가 네모 박스에 갇혀있음.
- **스탯 4개 + 스텝 4개 세트** — "37+ / 120+ / 80+ / 4.2K"와 "01 02 03 04"는 AI 랜딩 페이지 템플릿의 전형적 시그니처.
- **히어로 오른쪽 다크 카드의 정보 과밀** — 하이라이트 + 커피챗 설명 + 신규공고 + 활성 파트너 + 학교 로고 6개 + 불릿 3개가 한 박스에 들어감. 대시보드 같은 인상을 줌.
- **태그 pill 과다 사용** — 서울대/연세대/고려대/KAIST… 개발/서울/하이브리드… FOCUS 태그까지 pill이 너무 많이 반복됨.
- **팔레트는 방향성은 맞지만 톤이 어긋남** — 네이비(#1F3A6B 계열) + 베이지는 좋은 선택이지만, 카드에 흰색이 넘치고 다크 박스가 너무 어두워서 명도 대비가 거칠어짐.
- **한국어/영문 타이포가 단조로움** — 한 가지 산세리프만 사용. 세리프/산세리프 믹스가 없어서 "정보 전달" 느낌만 남고 "편집된 콘텐츠" 느낌이 부족함.

결론: 팔레트·방향은 유지하되 **카드 줄이고, 타이포를 무기로 쓰고, 여백을 늘리는** 방향으로 튜닝.

## 2. 레퍼런스 사이트 (중요도 순)

### ★ 핵심 레퍼런스 (반드시 확인)

**1. Ramp — https://ramp.com**
- 배울 점: 크림/오프화이트 배경 + 세리프 헤드라인 + 얇은 디바이더로 "건조하지만 고급스러운" 톤을 만드는 법.
- 구체적으로 가져올 요소: 히어로의 타이포 비중, 숫자·데이터를 카드 없이 구분선만으로 보여주는 방식.
- HireLink와 가장 팔레트가 비슷 — 지금 베이지/네이비를 유지하면서 가장 자연스럽게 참고 가능.

**2. Toss 채용 — https://toss.im/career/toss**
- 배울 점: 한국어 본문의 위계, Pretendard 기반 타이포 리듬, 차분한 블루 포인트 컬러 사용법.
- 구체적으로 가져올 요소: 긴 한국어 문장을 카드 없이 단락으로 보여주는 방법, 큰 여백, "말로 설명하는" 카피 톤.

**3. 29CM — https://www.29cm.co.kr**
- 배울 점: "감도 깊은 셀렉션"이라는 큐레이션 브랜딩이 시각적으로 어떻게 구현되는지.
- 구체적으로 가져올 요소: 큰 이미지/타이포 블록을 그리드로 배치하는 에디토리얼 레이아웃, 흑백+한 가지 포인트 컬러로 시선을 묶는 방식.
- HireLink의 "큐레이션 채용 플랫폼" 포지셔닝과 브랜드 로직이 가장 비슷.

### ◆ 보조 레퍼런스

**4. Mercury — https://mercury.com**
- 배울 점: 흰 배경에 아주 미묘한 파스텔 그라디언트, 세리프 헤드라인 + 산세리프 본문, 얇은 선 기반 정보 표현.
- 가져올 요소: 히어로 배경의 은근한 그라데이션, 숫자/지표를 카드 없이 보여주는 법.

**5. Readwise / Reader — https://readwise.io**
- 배울 점: 학생/지식워커가 "편안하게 읽을 수 있는" 프리미엄 감성. 세리프 중심 + 여백 + 따뜻한 크림 배경.
- 가져올 요소: "이번 주 하이라이트" 같은 섹션을 뉴스레터/매거진처럼 풀어내는 방법.

**6. The Browser Company (Arc) — https://thebrowser.company**
- 배울 점: 서사적인 글쓰기와 에디토리얼 레이아웃이 결합되는 방식, 브랜드 인격이 묻어나는 카피.
- 가져올 요소: 섹션 간 전환이 "페이지 넘기는" 느낌이 나도록 구성하는 기법.

**7. Handshake / Forage (학생 톤 참고)** — https://joinhandshake.com/students, https://www.theforage.com
- 주의: 두 곳 모두 HireLink 방향보다 더 캐주얼·친근. **톤을 그대로 가져오지 말고, 학생 메시지 카피만 참고.**
- 가져올 요소: 학생이 이해하기 쉬운 용어 선택, "가이드"를 해주는 어조.

## 3. HireLink 디자인 시스템 제안

### 3-1. 컬러 팔레트

현재 방향(네이비 + 베이지)을 유지하되 톤을 다시 잡기.

| 역할 | 제안 값 | 설명 |
|---|---|---|
| Background (기본) | `#FAF8F3` | 크림 오프화이트. 베이지보다 한 톤 밝고 따뜻함. |
| Background (보조) | `#FFFFFF` | 카드/분리 섹션용 순백. |
| Ink (본문) | `#151515` | 순 블랙 대신 거의 블랙. 종이 잉크 느낌. |
| Ink (보조 본문) | `#5C5C5C` | 설명/캡션. |
| Divider | `#E8E3D6` | 얇은 구분선. 베이지 톤. |
| Primary (네이비) | `#0F1F3D` | 기존보다 더 깊고 차분한 네이비. 현재 네이비는 너무 튐. |
| Accent (머스터드 골드) | `#C9A96E` | 기존 베이지 골드 유지. 태그·포인트용. |
| Accent (작은 포인트) | `#B84A2F` | 드물게 "핵심 강조"에만. Ramp가 오렌지를 쓰는 방식. |
| Success / 인증 뱃지 | `#2F5D3D` | 깊은 포레스트 그린. 학교 인증·검수 완료 표시. |

**사용 원칙**
- 배경은 90% 크림/화이트, 다크 네이비 섹션은 페이지당 **1개 이하**로 제한.
- 현재 히어로 오른쪽 다크 카드는 **제거**하거나, 쓰더라도 페이지에서 가장 중요한 한 곳에만.

### 3-2. 타이포그래피

한영 믹스 + 세리프/산세리프 믹스로 에디토리얼 위계를 만들기.

| 역할 | 제안 폰트 | 크기/굵기 |
|---|---|---|
| Display (히어로 헤드라인) | **Noto Serif KR / Nanum Myeongjo + Newsreader(영문)** | 64–88px, Medium/Semibold |
| H1 | Noto Serif KR + Newsreader | 40–48px, Medium |
| H2 | Pretendard + Inter | 28–32px, Semibold |
| H3 | Pretendard + Inter | 20–22px, Semibold |
| Body | **Pretendard + Inter** | 16–17px, Regular, line-height 1.65 |
| Caption / Eyebrow | Pretendard Mono / JetBrains Mono (대문자) | 12px, Medium, letter-spacing 0.08em |
| Numeric Display (통계) | Newsreader Italic | 48–64px, 숫자만 세리프+italic로 |

**핵심 원칙**
- **헤드라인은 세리프, 본문은 산세리프.** 이 차이만으로 AI 느낌이 크게 사라짐.
- Pretendard 기본. 영문은 Newsreader(세리프) + Inter(산세리프) 조합.
- 대문자 eyebrow("FOCUS", "추천 공고", "이용 흐름")에는 trackning 넓게. 현재도 이 패턴은 이미 쓰고 있어서 유지.

### 3-3. 레이아웃 & 여백

- **최대 콘텐츠 폭**: 1200px (현재는 가득 차 있는 느낌). 좌우 여백 충분히.
- **섹션 간 간격**: 120–160px (현재는 60–80px 수준, 너무 조밀).
- **12컬럼 그리드 엄격 적용**: 지금은 "카드 4개 일렬"이 반복되는데, 7+5 / 8+4 / 5+7 같은 비대칭 그리드를 섞으면 에디토리얼 느낌이 살아남.
- **카드는 세 번에 한 번만.** 나머지는 구분선 + 여백으로 섹션 구분.

### 3-4. 컴포넌트 규칙

**버튼**
- Primary: 네이비 배경 / 흰색 텍스트 / border-radius **4–6px** (현재 8px 이상은 "AI button"처럼 보임)
- Secondary: 투명 배경 + 1px 네이비 보더
- 높이 48px, 패딩 가로 24px

**카드 (쓸 때만)**
- border-radius 8px
- 그림자 대신 **1px 구분선** (`#E8E3D6`) 또는 은근한 상단 컬러 바
- 내부 패딩 32px 이상

**태그/pill**
- 현재처럼 남발하지 말 것. 한 화면에 최대 **5개 pill**.
- 라운드 8px (완전 캡슐형 말고), 배경 베이지 `#F0EBDB`, 텍스트 네이비.

**숫자 표시**
- 스탯 카드 제거. 대신 가로 인라인 + 세리프 이탤릭 + 얇은 디바이더.
- 예시: `37+ 인증 가능 학교 | 120+ 큐레이션 공고 | 80+ 주간 커피챗 | 4.2K 누적 멤버`

**이미지·일러스트**
- 아이콘 위주 X. 실제 학생/캠퍼스/오피스 사진을 흑백·세피아로 1–2장 섞으면 에디토리얼 느낌이 확 삼.
- 꼭 일러스트가 필요하면 얇은 선 드로잉 (outline, 1.5px).

## 4. 섹션별 구체 적용안

### 히어로
- 현재: 왼쪽 타이포 + 오른쪽 다크 대시보드 카드.
- 제안: **다크 카드 제거**. 전체 라이트 배경에 세리프 헤드라인만 + 그 아래 짧은 리드문 + CTA 2개 + **가로 인라인 스탯 줄**.
- 학교 로고는 그 아래 한 줄 그레이스케일 로고 바로 분리 ("인증 가능 대학: 서울대 연세대 고려대 KAIST POSTECH MIT Stanford Oxford").
- "이번 주 하이라이트" 정보는 별도 섹션으로 분리(매거진 목차처럼).

### "기존 채용 플랫폼과 다른 4가지"
- 현재: 4개 카드 일렬.
- 제안: 2×2 그리드 또는 **인덱스 스타일** — 왼쪽에 큰 섹션 타이틀, 오른쪽에 4개 항목이 세로로 나열되고 얇은 구분선만. 숫자 원형 배지 제거, 세리프 숫자만.

### "지금 주목할 포지션"
- 현재: 다크 네이비 카드 3개.
- 제안: 라이트 배경 카드 3개, 상단에 얇은 컬러 바 1px (Top Pick은 머스터드, New는 그린, Global은 네이비). 카드 내부는 크림 배경. 태그는 pill 대신 텍스트 + 구분점(`개발 · 서울 · 하이브리드`).

### "HireLink와 함께하는 방법"
- 현재: 3개 블록 + 다크 카드 1개.
- 제안: 전체 라이트. 다크 박스는 섹션 전체로 분리해서 "기업 파트너를 위한" 섹션으로 크게 풀기. 페이지당 다크 섹션 1개 규칙 지키기.

### "가입부터 채용까지 4단계"
- 현재: 왼쪽 숫자 배지 + 텍스트, 오른쪽 다크 카드.
- 제안: 타임라인 형식. 01-04를 세리프 대형 숫자(이탤릭)로 왼쪽에 크게, 그 옆에 텍스트. 오른쪽 다크 카드는 앞 섹션과 통합.
